# 🍽️ Restaurant Management System  
### Java Swing Application | Modern UI | Login • Sign Up • Menu • Orders

A fully designed **Java Swing Restaurant Management System** featuring a modern UI, user authentication, menu browsing, and order management.  
This project focuses on clean design, smooth user experience, and simple data storage using a **TXT file**.

---

## ✨ Features

### 🔐 Authentication System
- User Login  
- User Sign Up  
- Input validation (email format, password length, empty fields)  
- Error messages displayed under each field  
- Prevents duplicate email registration  
- Stores user data in `customers.txt`

### 🖥️ Modern User Interface
- Rounded buttons  
- Custom text fields  
- Shadow panels  
- Smooth panel switching animation  
- Gold & white theme  
- Responsive layout using GridBagLayout  

### 🍔 Restaurant Functionalities (Expandable)
- Menu page  
- Orders page  
- User profile  
- Manager login (optional)
